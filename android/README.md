# Flutter Android and iOS Configuration

This directory contains platform-specific files for Android and iOS.

## To generate complete platform files:

Run the following command to generate all necessary platform files:
```bash
flutter create .
```

This will create the complete Android and iOS project structure needed to build and run the app.

## Build Commands:

**Android:**
```bash
flutter build apk
```

**iOS (macOS only):**
```bash
flutter build ios
```

## Platform Requirements:

**Android:**
- Android Studio
- Android SDK (API level 21+)

**iOS:**
- macOS
- Xcode
- CocoaPods
